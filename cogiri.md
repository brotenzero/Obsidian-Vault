2 AI 모델 3가지

1>게스트형
	게스트용 (여행자 기능)  
① **AI 여행 플래너(기본)**  

- 입력: “4박 5일 오사카 여행, 미식 중심, 대중교통만 이용”
- 출력: 자동 생성된 일정표 + 숙소 추천 + 교통편
- 기술:
    - LLM (ChatGPT 계열) + 지도 API + TSP(최적 동선 알고리즘)
    - 사용자 취향 학습 (Collaborative Filtering)

② **AI 숙소 추천(기본)**  

- 사용자 행동, 리뷰, 위치, 기후 등을 고려한 맞춤형 추천
- “자연, 조용함, 애견동반” 등의 **감성 태그 기반 검색** 가능
- 기술: Embedding 기반 추천 / Matrix Factorization / Transformer Recommender

③ **이미지 기반 검색(확장)**  

- “이런 느낌의 숙소 찾아줘” → 사진 업로드
- Vision AI가 사진 특징(조명, 인테리어 스타일, 풍경)을 분석
- CLIP 모델 등 활용

④ **AI 후기 작성(확장)**  

- 키워드 입력 → 자동 리뷰 문장 생성
- 자연어 톤 자동 맞춤 (“격식”, “친근함” 등 선택 가능)


2>호스트형


**AI 숙소 설명문 자동 생성**  

- 호스트가 키워드(“바다 전망, 조용, 가족 여행”)와 사진 등 입력 
- AI가 감성적인 설명문 생성 

 **AI 가격 최적화 (Dynamic Pricing)**  

- 요일, 이벤트, 날씨, 수요 데이터를 AI가 분석
- “오늘 가격은 7% 인상 추천” 자동 알림
	
3>관리자형
**사기·이상 행위 탐지**  

- 반복 예약등 비정상 패턴 자동 감지
- Graph ML + Anomaly Detection 모델
- 관리자에게 리스크 점수로 시각화

## 🧠 1️⃣ “리팩토링(refactoring)”의 정의

> **리팩토링(Refactoring)** 이란,  
> **“코드의 외부 동작(기능)은 그대로 유지한 채 내부 구조를 개선하는 것”** 입니다.  
> — _Martin Fowler, 『Refactoring: Improving the Design of Existing Code』_

즉,

- **무엇을 하는지(기능)** 는 안 바꾸고
    
- **어떻게 하는지(구현)** 만 바꾸는 거예요.
