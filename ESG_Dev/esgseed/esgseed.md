발표 초반
실제 보고서 분량 예시보여줌 200페이지 분량 빠르게 스크롤
방어 : 완전 자동화 안되나? (실질적 데이터는 아직 사람의 검수가 필요함)
편집 단계 인력최소화 목적

기능1
지난 보고서 pdf 업로드하면 보고서 각 내용 섹션으로 분류하여 화면에 보여줌

기능2
화면에 보여진 섹션을 선택하면 섹션이 ai 컨텍스트에 저장되고
ai가 새로운 기준이 적용된 수정안을 먼저 제시하고 사용자와 대화하면서 내용 편집하고 버튼을 누르면 오른쪽 섹션에 저장

기능3
오른쪽 섹션 편집이 최종 완료 되고 프리뷰 버튼을 누르면 새로운 새로운 디자인 템플릿에 삽입되어 완성된 보고서 pdf파일 보여주고 pdf 파일 저장하기로 파일 다운
