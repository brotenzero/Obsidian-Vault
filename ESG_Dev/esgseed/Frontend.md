
현재 

```
frontend
```

 프로젝트의 개발 환경을 분석한 결과입니다.

**핵심 기술 스택:**

- **프레임워크**: **Next.js 14.2.33** (최신 안정 버전 중 하나)
    - ```
        src/app
        ```
        
         폴더 구조를 사용하는 것으로 보아 **App Router** 방식을 사용 중입니다.
- **언어**: **TypeScript 5.9.3** (정적 타입 시스템 적용)
- **스타일링**: **Tailwind CSS 3.4.1** (유틸리티 퍼스트 CSS 프레임워크)
- **패키지 매니저**: **pnpm** (빠르고 효율적인 패키지 관리 도구)

**주요 라이브러리:**

- **상태 관리**: 
    
    ```
    zustand
    ```
    
     (가볍고 직관적인 상태 관리 라이브러리)
- **아이콘**: 
    
    ```
    lucide-react
    ```
    
     (깔끔한 벡터 아이콘)
- **UI 유틸리티**:
    - ```
        class-variance-authority (cva)
        ```
        
        : 컴포넌트 스타일 변형 관리
    - ```
        clsx
        ```
        
        , 
        
        ```
        tailwind-merge
        ```
        
        : 클래스 이름 조건부 결합 및 충돌 방지
    - ```
        @radix-ui/react-slot
        ```
        
        : 컴포넌트 합성 유틸리티
- **HTTP 통신**: 
    
    ```
    axios
    ```
    
     (설치는 되어 있으나 현재 코드에서는 미사용)

**실행 환경:**

- **Node.js**: 18.0.0 버전 이상 필요 (
    
    package.json 설정 기준)
