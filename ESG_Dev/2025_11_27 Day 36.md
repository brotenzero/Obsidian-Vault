**  
BeautifulSoup4 (bs4)** 완전 정리 – 2025년 기준 한국 실무 필수 지식

| 항목          | 내용                                                                |
| ----------- | ----------------------------------------------------------------- |
| 정식 이름       | BeautifulSoup 4 (줄여서 bs4)                                         |
| 뭐하는 라이브러리?  | **HTML/XML 파싱 라이브러리** → requests로 가져온 웹페이지 소스를 예쁘게 분석해줌           |
| 가장 많이 쓰는 조합 | requests + BeautifulSoup4 = 정적 크롤링의 절대 왕                          |
| 설치 명령어      | pip install beautifulsoup4 (lxml 파서도 같이 깔자 → 빠름)                  |
| 추천 파서 순위    | 1. lxml (제일 빠름) 2. html.parser (기본, 설치 필요 없음) 3. html5lib (가장 관대) |
