
## 🧱 2️⃣ 주요 모듈별 구성

대부분의 인사프로그램은 다음과 같은 **핵심 기능 모듈**로 나뉩니다 👇

| 모듈                                     | 주요 기능                      | 사용 데이터                               |
| -------------------------------------- | -------------------------- | ------------------------------------ |
| **① 인사기본관리 (HR Core)**                 | 인사기본정보, 부서, 직급, 근속정보, 인사이동 | `employee`, `department`, `position` |
| **② 근태관리 (Attendance)**                | 출퇴근, 휴가, 근무시간, 연차관리        | `attendance`, `leave`                |
| **③ 급여관리 (Payroll)**                   | 급여계산, 수당, 공제, 세금처리         | `salary`, `tax`, `benefit`           |
| **④ 채용관리 (Recruiting)**                | 공고등록, 지원자관리, 평가, 채용결정      | `applicant`, `recruit_process`       |
| **⑤ 성과평가 (Performance)**               | KPI, OKR, 평가등급, 보상연계       | `performance`, `goal`, `review`      |
| **⑥ 교육/경력개발 (Learning & Development)** | 교육이력, 자격증, 내부교육추천          | `training`, `certificate`            |
| **⑦ 조직관리 (Organization)**              | 조직도, 팀 구조, 직무정의            | `org_chart`, `role`                  |
| **⑧ AI / 분석모듈 (AI Analytics)**         | 이직 예측, 성과 예측, 만족도 분석       | 통합된 HR 데이터                           |
| **⑨ 관리자/권한관리 (Admin)**                 | 접근제어, 역할별 권한, 로그기록         | `user`, `role`, `permissio`          |
